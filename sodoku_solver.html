Sudoku Solver in 5 minutes

About

Sudoku is a logical number placement puzzle. You have a 9x9 field and you need to fill each cell with numbers from 1 to 9 following the rules:

  • Each row has only distinct numbers
  • Each column has only distinct numbers
  • Each of 9 smaller 3x3 squares has only distinct numbers
Solving Sodoku requires a lot of patience and accuracy. However we can build an application that can solve Sudoku for us (or identify that there are no solutions).
