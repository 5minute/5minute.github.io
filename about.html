About this project

The goal of this project is to show people that it is possible to create mini-project in under 5 minutes.

Project's focus is:

  • Prove that programming is not that complex
  • Showcase quick 5 minutes mini-projects
  • Bring value to developers, especially those who are starting their journey
  • Use as less 3-rd party libraries and frameworks as possible
  • Share all solutions for free via GitHub repository
  • Share live-coding screen recording for building the solution

However, these items are not primary project's focus:

  • Clean code - often solutions are built with focus on speed rather on readability
  • Fully working solutions - for example while implementing classic game like "Snake" don't expect that it will be feature-complete

About me

My name is Sergey and I've been developing and architecting solutions for over 18 years for the clients all over the globe. Sometimes I feel that I need to break out from enterprise world and have some fun.

That's why I decided to start this project and indeed have some fun and share my knowledge.

The idea came when I recalled a story from my school time - a classmate challenged me if I can do a "Snake" game in 40 minutes. I did it within 25 or so minutes.
After 20 years I was explaining my kids how does "making computer programs" look like. And I also chose "Snake" game.
And then I thought - "Hm, can it be done faster? For example in 5 minutes?". I realized that there are lots of impressive things that can be coded in 5 minutes or less.

Despite working on this project in my free time as a hobby (or a pet?) project I have lot's of ideas what else can be coded in 5 minutes.
